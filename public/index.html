TOYS API

Through these requests it is possible to register and connect , admin can get all users list , all user can add toys , update and delete him toys,
admin can update and delete all user's toy , you can show all toys (perPage,page,sort,reverse).

Users

Get
/users/usersList
Get list of all users - by token admin.
POST
/users
Add a new user to the data base.
                                Body
                                
                                    {
                                        "name":"David",
                                        "email":"David@gmail.com",
                                        "password":"123456"
                                    }
POST
/users/login
User login, if the email and password are correct the server will respond with a token, 
                            Body
                            
                                {

                                }

Toys

Get
/toys
Get a list of all the toys, will present 10 results per page.
Get
/toys/search?s=blocks
search for a toy by searching follwing name or descraption after the "s=", it will present 10 results per page.
Get
/toys/category/Building
Get a list of all the toys in toys category, it will present 10 results per page
Get
/toys/single/:_idToy
Get a toy by id
Get
/toys/price/?min=10&max=30
Get a list of the toys by min & max , it will present 10 results per page.
POST
/toys
Add a new toy to the data base, You need a token for this query
                                Body
                                
                                    {
                                     "name": "Doll",
                                    "description": "Classic doll with various accessories and outfits",
                                    "category": "Dolls",
                                    "image": "url_to_doll.jpg",
                                    "price":40 
                                    }
POST
/toys/{editId}
Update a toy in the data base,You need a token for this query.
                                Body
                                
                                    {
                                        "name": "Doll",
                                       "description": "Classic doll with various accessories and outfits",
                                       "category": "Dolls",
                                       "image": "url_to_doll.jpg",
                                       "price":35 
                                       }
.
DELETE
/toys/{delId}
Delete a toy from the data base, You need a token for this query.
